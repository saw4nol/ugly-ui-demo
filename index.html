The Ugliest UI I Could Design

A deliberately bad UI and notes on how to improve it.

Screenshot of the "ugly" UI

What’s wrong with this UI?

  • Horrible color contrast and clashing hues.
  • Poor layout with misaligned elements and no spacing system.
  • Unreadable fonts (too many font sizes and styles used randomly).
  • Non-semantic HTML — accessibility ignored (no ARIA, no landmarks).
  • Confusing interaction patterns and unclear affordances.

What I Would Do to Improve It

  1. Use consistent fonts, colors, and spacing to make everything look organized.
  2. Fix the color contrast so text is easier to read.
  3. Make sure all buttons and links are easy to reach and work with a keyboard.
  4. Use a simple, responsive layout that looks good on phones and computers.
  5. Write clear labels and buttons so users instantly know what to do.
